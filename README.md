## <span style="color:#bc6fa8">🎯 What is ETL?</span>

**<span style="color:#bc6fa8">E</span><span style="color:#bc6fa8">T</span><span style="color:#bc6fa8">L</span> <span style="color:#fff">|</span> <span style="color:#bc6fa8">E</span><span style="color:#fff">nhanced </span><span style="color:#bc6fa8">T</span><span style="color:#fff">raveler's </span><span style="color:#bc6fa8">L</span><span style="color:#fff">og</span><span style="color:#bc6fa8">!</span>** <span style="color:#e67e23">overlays clean, themed progress bars on every row of the Monthly Activities frame and adds a minimap launcher that opens the journal directly to the Traveler's Log tab. Hide completed rows, monitor live updates as you earn progress, and skip the extra clicks needed to navigate Blizzard's UI.</span>

### <span style="color:#bc6fa8">🔥 Why Choose ETL?</span>
- **<span style="color:#2dc26b">📊 Visual Progress:</span>** <span style="color:#e67e23">Themed progress bars on every Monthly Activity row</span>
- **<span style="color:#bc6fa8">⚡ One-Click Access:</span>** <span style="color:#e67e23">Minimap button opens straight to the Traveler's Log</span>
- **<span style="color:#b96ad9">🙈 Hide Completed:</span>** <span style="color:#e67e23">Optional setting to collapse finished activities</span>
- **<span style="color:#4ecdc4">🔄 Live Updates:</span>** <span style="color:#e67e23">Bars refresh automatically as you make progress</span>
- **<span style="color:#ff6b6b">💬 Active Support:</span>** <span style="color:#e67e23">Join our Discord for instant help!</span>

<a id="features"></a>
## <span style="color:#bc6fa8">✨ Features</span>

<div align="center">

| Feature | Description |
|---------|-------------|
| 📊 **Progress Bars** | Themed bars overlay every Monthly Activity row with current/total progress |
| 🎨 **Brand Color** | Bars use the ETL palette by default — clean, distinctive, easy to read |
| ⚡ **Minimap Launcher** | One click opens the Encounter Journal straight to Monthly Activities |
| 🙈 **Hide Completed** | Toggle finished activities so the list shows only what's left |
| 🔄 **Live Refresh** | Bars update automatically when activity progress changes |
| 🧱 **RGX-Framework Native** | Built on the shared RGX-Framework — no LibStub bloat |
| 💾 **Persistent Settings** | Minimap angle, visibility, and bar preferences saved between sessions |
| 🛡️ **Combat Safe** | Skips opening the journal during combat lockdown |

</div>

<a id="quick-start"></a>
## <span style="color:#bc6fa8">🚀 Quick Start</span>

1. **<span style="color:#2dc26b">Install</span>** <span style="color:#bc6fa8">E</span><span style="color:#bc6fa8">T</span><span style="color:#bc6fa8">L</span> <span style="color:#e67e23">from your preferred platform (RGX-Framework is required)</span>
2. **<span style="color:#4ecdc4">Launch</span>** <span style="color:#06c">World of Warcraft</span>
3. **<span style="color:#b96ad9">Click</span>** <span style="color:#e67e23">the ETL minimap icon — opens straight to Monthly Activities</span>
4. **<span style="color:#bc6fa8">Progress bars appear automatically</span>** <span style="color:#e67e23">on every activity row!</span>
5. **<span style="color:#2dc26b">Type</span>** `/etl help` <span style="color:#e67e23">to see all available commands</span>

---

<a id="command-reference"></a>
## <span style="color:#bc6fa8">📋 Command Reference</span>

<span style="color:#e67e23">Use</span> <span style="color:#2dc26b">`/etl`</span> <span style="color:#e67e23">followed by</span><span style="color:#3598db">:</span>

<div align="center">

| Command | Description |
|---------|-------------|
| `/etl` | Open the Traveler's Log directly |
| `/etl help` | Show all available commands |
| `/etl status` | Show current addon state and visible row count |
| `/etl bars on` / `bars off` | Enable or disable progress bars |
| `/etl completed on` / `off` | Hide or show completed activity rows |
| `/etl welcome` | Toggle the login welcome message |
| `/etl debug` | Toggle ETL debug output |
| `/etl refresh` | Re-scan visible Traveler's Log rows |
| `/etl icon on` / `icon off` | Show or hide the minimap icon |
| `/etl version` | Show ETL version |

</div>

## <span style="color:#bc6fa8">📥 Installation</span>

1. **<span style="color:#2dc26b">Install RGX-Framework</span>** <span style="color:#e67e23">first — ETL requires it</span><span style="color:#3598db">:</span>
   - [<span style="color:#8B1538">RGX-Framework on GitHub</span>](https://github.com/DonnieDice/RGX-Framework)

2. **<span style="color:#2dc26b">Download ETL</span>** <span style="color:#e67e23">from your preferred platform</span><span style="color:#3598db">:</span>
   - [<span style="color:#ff6b6b">CurseForge</span>](https://www.curseforge.com/wow/addons/enhanced-travelers-log)

3. **<span style="color:#4ecdc4">Extract</span>** <span style="color:#e67e23">to your</span> <span style="color:#06c">WoW</span> <span style="color:#e67e23">AddOns directory</span><span style="color:#3598db">:</span>
   - <span style="color:#e67e23">**Retail**: `World of Warcraft/_retail_/Interface/AddOns`</span>

4. **<span style="color:#b96ad9">Restart</span>** <span style="color:#06c">WoW</span> <span style="color:#e67e23">and enable in AddOns menu</span>

## <span style="color:#bc6fa8">🔧 Troubleshooting</span>

**<span style="color:#ff6b6b">Bars not showing?</span>**
- <span style="color:#e67e23">Confirm RGX-Framework is enabled in your AddOns list</span>
- <span style="color:#e67e23">Run</span> `/etl status` <span style="color:#e67e23">to check the active state</span>
- <span style="color:#e67e23">Try</span> `/etl refresh` <span style="color:#e67e23">to re-scan visible rows</span>

**<span style="color:#ff6b6b">Minimap click opens to the wrong tab?</span>**
- <span style="color:#e67e23">Update to the latest release — older versions had a tab navigation regression</span>
- <span style="color:#e67e23">Try</span> `/reload` <span style="color:#e67e23">to refresh the UI</span>

**<span style="color:#ff6b6b">Still having trouble?</span>**
- <span style="color:#e67e23">Join our</span> [<span style="color:#7289da">Discord</span>](https://discord.gg/hK9N3esnce) <span style="color:#e67e23">for instant support</span>
- <span style="color:#e67e23">Open a</span> [<span style="color:#ff6b6b">GitHub Issue</span>](https://github.com/DonnieDice/EnhancedTravelersLog/issues)
