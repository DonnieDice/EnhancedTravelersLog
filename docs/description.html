# 🎯 RGX | Enhanced Traveler's Log! ## ![](https://github.com/donniedice/SimpleQuestPlates/blob/main/media/kiwi.gif?raw=true) RGX Mods - RealmGX Community Project *** ## 🎯 Overview **RGX | Enhanced Traveler's Log!** overlays clean, themed progress bars on every Monthly Activity row in World of Warcraft's Encounter Journal and adds a minimap launcher that opens straight to the Traveler's Log tab. Watch your monthly progress fill in real time, hide completed rows, and skip the extra clicks. ![RealmGX Kiwi](https://github.com/donniedice/SimpleQuestPlates/blob/main/media/kiwi.gif?raw=true) **The Kiwi Says:** "Every bar tells a story! Bwwiiiee." *** ## ✨ Features 📊 Progress Bars - Themed bars overlay every Monthly Activity row with current/total progress! ⚡ Minimap Launcher - One click opens the Encounter Journal straight to Monthly Activities. 🙈 Hide Completed - Optional setting collapses finished activities so the list shows only what's left. 🔄 Live Updates - Bars refresh automatically as you make progress in-world. 🧱 RGX-Framework Native - Built directly on the shared RGX-Framework — no LibStub bloat, no fallbacks. 🛡️ Combat Safe - Skips opening the journal during combat lockdown so it never breaks your fight. 💾 Persistent Settings - Minimap angle, visibility, and bar preferences saved between sessions. 🎨 Brand Color Theme - Bars use the ETL palette by default — clean, distinctive, easy to read. **Required Dependency:** RGX-Framework must be installed and enabled. *** ## 📋 Slash Commands The addon can be managed using the following slash commands:
/etl # Open Traveler's Log
/etl help # Show all available commands
/etl status # Show current addon state
/etl bars on/off # Enable or disable progress bars
/etl completed on # Hide completed activity rows
/etl refresh # Re-scan visible Traveler's Log rows
/etl icon on/off # Show or hide the minimap icon
